SHELL-TO-GO MIGRATION (long-term)
================================================================================

Goal: eliminate shell scripts where possible, replace with Go binaries.
Benefits: no quoting bugs, no word-splitting, no A&&B||C pitfalls,
cross-platform, testable, type-safe.

  [x] test-coqprime-vm.sh → done (tools/cmd/test-coqprime created in v0.5.14)
  [x] release-package.sh + lib-release.sh → done (tools/cmd/release created in v0.5.14)
