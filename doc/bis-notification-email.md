**Cryptographic Algorithms Implemented:**
- AES-256 (FIPS 197)
- AES-256-GCM (NIST SP 800-38D)
- SHA-256, SHA-512 (FIPS 180-4)

- ML-KEM-768 (FIPS 203, post-quantum key encapsulation)
- X25519 (RFC 7748, Diffie-Hellman key agreement)
- Ed25519 (RFC 8032, digital signatures)
- HMAC (RFC 2104)
- HKDF (RFC 5869)
- ChaCha20 (RFC 8439)
- Poly1305 (RFC 8439)
- Signal Protocol (X3DH, Double Ratchet, PQXDH hybrid)
- Noise_IK handshake protocol

under 15 CFR 734.3(b)(3) (publicly available technology and software).

## Notes

- This is a **notification**, not a license request. BIS does not grant or deny — they log the notification.
- File this **before** any public release or distribution outside the US.
- **Retain a copy** of the sent email and any confirmation/auto-reply as evidence of filing.
- The notification covers all current and future versions of the publicly available source code at the specified repository URL.
- If the repository URL changes, file a new notification.
- This follows the same process used by Signal Messenger (Open Whisper Systems) for their encryption source code.
